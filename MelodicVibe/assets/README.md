# Assets Directory

Direktori ini berisi semua aset visual dan media untuk aplikasi MelodicVibe.

## Struktur Folder

### /icons
- Icon aplikasi
- Icon fitur
- Icon navigasi
- Format: PNG (transparent background)

### /screenshots
- Screenshot aplikasi untuk Play Store
- Screenshot untuk marketing
- Format: PNG/JPG (1080x1920px)

### /feature_graphics
- Banner Play Store
- Promotional graphics
- Format: PNG/JPG (1024x500px)

### /videos
- Video promosi
- Tutorial
- Format: MP4 (H.264)
